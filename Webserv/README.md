How to use:
1 - Download parent folder of the project or clone it using git
	$> git clone https://github.com/taoufiqjmari/1337-Common-Core.git

2 - On Terminal, go to project directory
	$> cd Webserv

3 - Make the executable
	$> make

4 - Configure a configuration file for the server (default.config is already available)

4 - Run
	$> ./webserv default.config
	// Go to a browser or Postman and request the routes you configured (e.g. localhost:8000/)
